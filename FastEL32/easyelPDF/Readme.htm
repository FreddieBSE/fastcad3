Installationen är klar

easyelPDF är installerad som en ny skrivare.

 

För att skapa PDF-filer

Använd Arkiv -> Skapa PDF för att skapa PDF-filer automatiskt. Du kan också använda Akriv -> Skriv ut för att få fler inställningsmöjligheter.

 

Avinstallera easyelPDF:

 

Öppna kontrollpanelen, välj avinstallera ett program och dubbelklicka på easyelPDF.
